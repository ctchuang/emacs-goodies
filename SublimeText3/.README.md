- On OS X

```console
$ cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User
$ ln -s <path_to_emacs_goodies>/SublimeText3/* .
```

- On Ubuntu

```console
$ cd ~/.config/sublime-text-3/Packages/User
$ ln -s <path_to_emacs_goodies>/SublimeText3/* .
```

> This won't create symbolic links to hidden files in this folder.
